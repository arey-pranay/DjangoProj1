### install uv
`powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`

### create a project
`uv venv`

### activate virtual environment
`.venv\Scripts\activate`

### install Django
` uv pip install Django`
